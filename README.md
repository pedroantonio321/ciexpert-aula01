The file gen-prj.sh is a project creator. The usage is: ./gen-prj.sh <projectName>
There are a few projects created inside of the folder prj/:
calc: Make a few operations like add, multiply, subtract and division
prj: A simple AND Gate
RAM: A simple project trying to make a RAM(Still in progress)
ROM: A simple project trying to make a ROM(Still in progress)

All these projects were made using SystemVerilog, but they were not tested with a simulator yet.
